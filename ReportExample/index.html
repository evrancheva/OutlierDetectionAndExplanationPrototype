Report on Remarkable Changes in Employees' Data for November - December


Summary

The aim of this report is to extract and explain changes for the employees in the company of ASR (www.asrnederland.nl) when comparing data for two months (November - December).

Terms

An attribute (also known as feature) is a property that is shared among different objects. In this report, an attribute is a property that is shared among employees. There are different types of attributes - numerical and categorical.
  • A numerical attribute is the one that can take any numeric value. Example of numerical values are base salary, relative salary position and etc.
  • A categorical attribute is the one that can take on fixed number of possible values. The categorical attributes are also classified as nominal and ordinal.
    • An ordinal attribute is an attribute in which there is a clear ranking of values (e.g. employee_grade). A step in ordinal attributes is a number that describes the transition in the attribute. For example, the attribute performance_status has values low, medium, high. If the performance status of employee changes from low to medium, the step is 1. If it changes from low to high, the step is 2.
    • A nominal attribute is an attribute that is used to label objects (e.g. title, business unit)
      • .
An extreme value (also known as an outlier) is a value that is very different from the others. In this report, an extreme value for numerical attributes is every value that is either very low or very high compared to the others. An extreme value for categorical attributes is every value in which there are a few overall frequencies. For example, an employee that has title CEO is an extreme value because there are just a few people that have the same title.
A change is every alteration of values when comparing data from two time periods. In this report, when comparing data for November - December. After detecting the changes, we use the values from the second month and identify if the values are extreme, interesting or normal..
A remarkable change is considered:
  • Every employee with extreme values in multiple attributes
  • Every employee with extime value in one attribute
  • Every employee with changes that exceed certain threshold
A measure of association is a measure that shows how well two attributes are related. It has a value between 0 and 1, where 0 shows that the attributes are not related at all and 1 shows that the attributes are perfectly related. Perfectly related means that there is a relationship between the attributes and the change in one of the attributes might have influence on the other.
